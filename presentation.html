Still image saved → POST /alerts/upload-evidence → POST /alerts/ → WS notification pushed to all dashboard clients.

Video Evidence — 5 sec clip (2s pre + 3s post)

User Resolves Alert → 5-min Label Mute

PATCH sets is_resolved. Backend mutes that violation label for 5 minutes, AI service polls and suppresses duplicate alerts.

AI Service · detector.py

Hybrid Detection Pipeline

YOLO for spatial PPE checks · Grok Vision LLM for semantic scene understanding.

1

YOLO Inference (Synchronous)

Model: safe.pt · Resolution: 416px · Person conf >0.4 · PPE conf >0.2

2

PPE Association via IoU

Per-person: checks helmet/vest/harness overlap (>0.1 IoU). Missing items → "Violation: No Helmet, Vest"

3

Grok-2 Vision LLM (Async Thread)

Frame encoded as base64 JPEG → POST to api.x.ai/v1. Throttled: max 1 req/1.5s

4

Merge Logic

IoU >0.3 → LLM may override YOLO. IoU <0.3 + conf >0.4 → new LLM detection appended.

5

Annotate & Stream

Bounding boxes drawn. YOLO=red/green, LLM=magenta. JPEG→WebSocket.

🧠 Grok Vision Detects

No Helmet / Vest / Harness
Unauthorized / Restricted Zone
⚠️ Danger Zone (Excavator swing)
Fall Risk (unprotected edge)
Phone / Smoking in vehicle zone
Health Emergency (person down)
Fire / Smoke detection

⏱ Alert Cooldown

10-second per-label cooldown prevents DB flooding. Each unique violation class tracked independently on the detector object.

AI Service · main.py

Snapshot Recorder

Automatic DVR-style incident clip capture with pre/post roll buffering.

📼 Recording Pipeline

Circular Pre-Roll Buffer

Deque holds last 2 seconds of frames at 30 FPS = 60 frames max. Always rolling, overwrites oldest.

Trigger: Violation Detected

recorder.start_recording() called. Buffer contents copied as clip start. Frames_left = 90 (3s × 30fps).

Post-Roll Recording

Next 3 seconds of live frames appended to clip. Total clip = 5 seconds contextual evidence.

H.264 MP4 Saved & Uploaded

Written with avc1 codec for browser playback. Filename: snapshot_{cam}_{ts}_{uuid6}.mp4

4-Step Alert Upload Flow

POST /alerts/upload-evidence → returns /media/filename

Create alert record

POST /alerts/ with image_url + all metadata

Upload MP4 clip

POST /alerts/upload-evidence → returns /media/snapshot.mp4

Attach video to alert

PATCH /alerts/{id} with video_url
